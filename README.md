# Automobile Sales Data Dashboard 🚗📊

This project was built as the capstone for the IBM Data Visualization with Python certification. It analyzes historical automobile sales data to uncover trends during economic recessions versus non-recession periods.

## Features
* **Interactive Dashboards:** Built using Plotly and Dash.
* **Exploratory Data Analysis:** Includes Pandas and Seaborn visualizations in a Jupyter Notebook.
* **Key Metrics Analyzed:** Sales volume, advertising expenditure, unemployment rates, and vehicle types.

## How to Run This Project
1. Clone this repository to your local machine.
2. Install the required libraries: `pip install pandas dash plotly`
3. Run the application: `python DV0101EN-Final_Assign_Part_2_Questions.py`
4. Open your web browser and go to `http://127.0.0.1:8050/`
